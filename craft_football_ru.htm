Парсилка футбольного сайта.

Друг попросил сделать штуку, которая выдирает данные с сайта и забрасывает их в эксель: нужно копировать в эксель поле за полем из статистики к матчуF и потом строить аналитику в экселе. Он устал он этой обезьяньей работы и попросил меня автоматизировать заполнение исходных таблиц для анализа. Я погуглил и просмотрел код страницы. Программа берет адрес страницы из буфера обмена, забирает его содержимое и находит нужное.
Особенности:
  • нет многопоточности, т.к. нет больших вычислений
  • русский интерфейс
Фишки:
  • Можно использовать архитектуру приложения для создания собственной программы по парсингу сайтов
  • Используется паттерн "стратегия"
